# Playlists

Playlists are a top-level tree item in the Library pane, under "All Tracks"

When Playlists are highlighted in the Libary pane, users see a list of playlists in the right pane. They can be sorted by either alphabetical order, or recency of use (default)


## Adding Songs To a Playlist

When a song is highlighted in the right pane (in any view - playlists, etc) it can be added to a playlist with a hotkey. This opens a modal allowing the user to select which playlist it should be added to.

Modal contents
- Recent Playlists
  - Show the 3 most recently edited playlists (note, there may be less than 3 extant playlists)
  - Show a list of all playlists, which offers a find-as-you-type filter for quick access

If the selected song already exists in the playlist, ask the user to confirm if they want to add the duplicate

## While a Playlist is in the right pane

Hotkeys to move selected track up or down

Track or entire playlist can be played / enqueued just like other song views

Hotkeys to:
- Duplicate playlist (ask user for new playlist's name)
- Delete playlist (ask for confirmation)
